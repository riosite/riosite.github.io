Web｜職業訓練校課題

ウェディングフォトスタジオサイト「Kyouritz Photo Studio」（架空）

URL : --------------------------------

制作概要

職業訓練校課題で、ウェディングフォトスタジオサイトの架空サイトを制作しました。

依頼内容

・綺麗で落ち着いた、ラグジュアリーな雰囲気。
・撮影や試着会の予約に繋がるような導線を確保。
・無料ウェディングドレス試着会のバナーを作成。
・モバイルファーストで作成。

目的

利用者の集客、試着会の宣伝

ターゲット

20～30代の男女

情報設計

デザイン

指定されたロゴを使用しているため、ロゴに合わせた配色でデザインしました。
ベースカラーにはペールホワイト、アクセントカラーはダークな色合いでまとめ、上品さとラグジュアリーな雰囲気を演出しました。
また、撮影や試着会の予約に繋がるよう、各所に資料請求のボタンや、試着会のバナーを設置しました。
依頼通りモバイルファーストで作成し、PC表示の際は操作性を考慮して、余白にナビゲーションと資料請求ボタンを配置しました。

制作範囲

ワイヤーフレーム / デザイン / コーディング

使用ツール

Photoshop / Dreamweaver / Visual Studio Code

制作期間

5日間

制作時期
